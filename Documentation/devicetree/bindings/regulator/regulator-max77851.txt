Regulator DT binding for MAX77851 Power management IC from Maxim Semiconductor.

- maxim,out-voltage-range:	Output Voltage Range
					enabled/disabled when system is in
					active state.  Valid values are:
					- MAX77620_FPS_SRC_0,
						FPS source is FPS0.
					- MAX77620_FPS_SRC_1,
						FPS source is FPS1
					- MAX77620_FPS_SRC_2 and
						FPS source is FPS2
					- MAX77620_FPS_SRC_NONE.
						Regulator is not controlled
						by FPS events and it gets
						enabled/disabled by register
						access.
					Absence of this property will leave
					the FPS configuration register for that
					regulator to default configuration.

- maxim,fps-user-setting-enable: using default setting (OTP/SHADOW)

- maxim,regulator-dvs-mode-enable: DVS Mode Enable
					the regulator get disabled when master
					FPS input event set to LOW.
					Valid values are 0 to 7.
					This is applicable if FPS source is
					selected as FPS0, FPS1 or FPS2.

- maxim,regulator-dvs-voltage: VOUT0/VOUT1 Voltage array

- maxim,pu-slot:	FPS Power-Up Slot
- maxim,pd-slot:	FPS Power-Down Slot
- maxim,slpx-slot:	FPS Sleep Exit Slot
- maxim,slpy-slot:	FPS Sleep Enter Slot
					Valid values are 0 to F.
